# Quick Start - Building Installers

## macOS
```bash
cd installers/mac
./build_installer.sh
```
Output: `output/KalshiPredictor-1.0.0.pkg`

## Windows
```powershell
cd installers\windows
.\build_installer.ps1
```
Output: `output\KalshiPredictorSetup-1.0.0.exe`

## Before Building
1. Update version in build scripts
2. Run `installers/common/convert_icons.sh` (first time only)
3. Test the app with `./install.sh && ./run_app.sh`

## What Gets Included
- All backend code
- All frontend code
- Dependencies (installed during setup)
- Public key for license verification
- App icon
- Installation scripts

## What Users Need
- Python 3.8+
- Node.js 16+
- A valid license key (generated by you)

## Testing
Always test on a clean machine before distributing!

See BUILD_GUIDE.md for detailed instructions.
