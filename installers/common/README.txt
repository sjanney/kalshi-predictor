# Kalshi Predictor - User Guide

## Welcome to Kalshi Predictor!

Thank you for installing Kalshi Predictor. This guide will help you get started.

## First Launch

1. **Activate Your License**
   - When you first launch the application, you'll be prompted to enter a license key
   - Paste the license key provided to you
   - Click "Activate"

2. **Configure API Keys** (Optional)
   - If you want to use live market data, you'll need Kalshi API credentials
   - Go to Settings → API Configuration
   - Enter your Kalshi API key and secret

## Using the Application

### Dashboard
- View upcoming games and predictions
- See model confidence scores

### Analytics
- Deep dive into prediction models

### Settings
- Manage your license
- Configure API keys
- Adjust preferences

## System Requirements

### macOS
- macOS 10.15 (Catalina) or later
- Python 3.8 or later
- Node.js 16 or later
- 4GB RAM minimum
- 500MB free disk space

### Windows
- Windows 10 or later
- Python 3.8 or later
- Node.js 16 or later
- 4GB RAM minimum
- 500MB free disk space

## Troubleshooting

### Application won't start
1. Ensure Python and Node.js are installed
2. Check that ports 8000 and 5173 are not in use
3. Try running the install script again

### License issues
- Verify your license key is correct
- Check your internet connection
- Contact support if the issue persists

### Performance issues
- Close other applications to free up memory
- Check your internet connection speed
- Ensure your system meets minimum requirements

The application will notify you when updates are available. To update:
1. Download the latest installer
2. Run the installer (it will upgrade your existing installation)
3. Your license and settings will be preserved
