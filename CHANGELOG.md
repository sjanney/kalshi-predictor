# Changelog

All notable changes to this project will be documented in this file.

## [3.0.0] - 2025-11-24

### Added
- **Production Configuration**: Centralized configuration via `config.py` and environment variables.
- **Error Boundaries**: React Error Boundary for graceful UI failure handling.
- **Logging**: Structured logging replacing print statements in backend services.
- **Security**: Enhanced CORS configuration and Electron security settings (context isolation, disabled node integration).

### Changed
- **UI Refinement**: Improved Dashboard stats spacing, loading skeletons, and typography.
- **Game Card**: Refined badge positioning and visual hierarchy.
- **Analytics**: Enhanced chart spacing and tooltips.
- **Architecture**: Consolidated backend services (marked legacy services as deprecated).

### Removed
- **Unused Code**: Removed `VirtualizedGameList` component and unused imports.

## [2.0.0] - Previous Version
- Initial release of Enhanced Prediction Engine.
- Added Signal Engine and Insights Generator.
- Integrated Kalshi market data.
